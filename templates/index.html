nightskycam: camera local server

Visit https://192.124.27.162/snapshots to pause the nightskycam service first (this functionality will soon be integrated in this page).
  • before taking picture, activate the adapter
  • it takes around 5 seconds for the adapter to activate
  • the capture will try to activate the adapter if it has not been activated, but related error messages may be silent
  • close the adapter to fix the focus at the value of the latest captured picture
  • try to close and re-activate the adapter if captures fail
  • (re)activating the adapter will unfocus the system, even if no new image is captured
